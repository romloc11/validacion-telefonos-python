# Validación masiva de números telefónicos con Python

Script desarrollado en Python que permite validar de forma masiva números telefónicos utilizando una API externa.

El proceso identifica:
- Si el número es válido
- Tipo de línea (móvil o fijo)
- Carrier
- País

Los números se cargan desde un archivo Excel y los resultados se exportan automáticamente a otro archivo Excel.

## Problema que resuelve
Antes, la validación se realizaba de forma manual número por número, lo que consumía varias horas de trabajo operativo.

Este desarrollo permite validar cientos o miles de números en minutos, automatizando completamente el proceso.

## Tecnologías utilizadas
- Python
- Pandas
- Requests
- APIs REST
- Excel

> Nota: Se requiere una API Key válida para ejecutar el script.
